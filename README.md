# 🤖 AI Recruitment System

An AI-powered Recruitment System developed using **Python**, **Streamlit**, **PostgreSQL**, and **SQLAlchemy**. The system automates resume screening, candidate evaluation, resume scoring, and recruitment management through separate dashboards for **HR** and **Candidates**.

---

## 📌 Features

### 👨‍💼 HR Dashboard

- Secure HR Login (Predefined Credentials)
- Dashboard Statistics
- Post New Job Openings
- Manage Posted Jobs
- View Total Applicants
- Close Job Openings
- View Candidate Details
- Resume Score
- Top 3 Candidates
- Selected Candidates
- Rejected Candidates
- AI Chatbot for Candidate Search

---

### 👨‍🎓 Candidate Dashboard

- Candidate Registration
- Candidate Login
- View Available Jobs
- Apply for Job
- Upload Resume (PDF/DOCX)
- AI Resume Parsing
- Automatic Information Extraction
- Resume Score
- Application Status

---

## 🧠 AI Features

- Resume Parsing
- OCR Support (Image-based Resumes)
- Candidate Information Extraction
- Resume Scoring (0–100)
- Resume Skill Matching
- AI Recommendation
- HR AI Chatbot
- Skill-Based Candidate Search

---

## 📊 Resume Score

The AI calculates a resume score based on:

- Skills Match
- Education
- Experience
- Required Technologies
- Resume Completeness

Score Range:

- 0 – 100
